Explanation:
Color names are basic colors already given in VS Code, like red, white, blue, and green. We can use them just by writing their names.
HEX colors are made with six letters or numbers, for example #172cak, fge345. Every color has a different HEX code, and it is used to show colors on websites.
RGB colors are made with three numbers red blue and green, like rgb(195, 210, 655). This gives us more control to make many different shades of colors.
